Introduction
This is a pure Java implementation of Configuration Tool. The program is tested on Redhat
7.3, 8, 9 and Solaris 8 with Sun JDK 1.4.0, 1.4.1 and 1.4.2.

Requirements
1. JRE 1.4.0 and up is required to run this program and JRE 1.4.2 is recommended
2. JRE 1.5 may work but it requires a different version of Bouncy Castle library
3. The unrestricted policy files for JRE has to be installed

Installation
1. untar gapslite.tar.gz to a directory
2. edit bin/encode.sh script and change GAPSLITE_HOME and JAVA_HOME variables
3. edit config/gaps.cfg to set license information
